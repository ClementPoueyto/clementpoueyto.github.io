Bonsoir bonsoir, blaud et facq.

Il reste moins d'un mois avant la fin, votre mission est de :

- Créer des niveaux reprenant le style du fichier Arena.
	Le niveau doit avoir son propre style, ses propres difficultes afin qu'il soit différents des autres.
Chaque niveau possède son propre thème : voir fichier "boiteaidee.txt"
Le theme étant " The Place To Be " vous pouvez faire le theme qui vous convient tant qu'il respecte la consigne.

Pour créer un niveau il faut obligatoirement : 
	un mesh "limit" qui correspond au point qui fera réapparaitre le joueur s'il est franchi.
	un mesh "end" qui fera revenir le joueur au menu. Ce mesh correspond à la fin d niveau
	un attribut "spawn" qui correspond a la position de réapparition du joueur. 
	Une musique personnalisée du niveau, ainsi que des bruitages si besoins. (La musique doit être créée par nous, impossible de prendre sur L'INTERNET.)

	Un attribut speed correspondant à la vitesse de déplacement du personnage; ( Vecteur)
	
	une liste de mesh "listOfMesh" regroupant tous les mesh du niveau.
Cette liste est utilisée pour effacer tous les composants une fois le niveau fini (on économise la RAM|ROM de l'utilisateur). Il est donc important de push  à l'interieur tous les elements créés.

Ne pas oublier d'activer les collisions ainsi que les physicsImpostor.


Il faut aussi créé les textures de chaque mesh
		créé la texture du joueur(sphere) propre a chaque niveau


-créer une (tres bonne) musique pour le menu.

- ameliorer esthetiquement le menu.
